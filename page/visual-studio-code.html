Visual Studio Code

Microsoft tarafından geliştirilen Gelişmiş bir metin editörü...

Visual Studio Code, hata ayıklama özellikleriyle, gelişmiş web ve bulut uygulamaları üstünde kodları düzenlemeye, yeniden tanımlamaya ve optimize etmeye yarar. Visual Studio Code tamamen ücretsiz olup, dilediğiniz gibi kullanabilir, kodlarını inceleyebilir ve kendi ihtiyaçlarınıza göre değişim yapabilirsiniz.

Kurulum:

Aşağıda ki komutu girerek uygulamayı derleyip inşa edin:

sudo pisi bi https://raw.githubusercontent.com/pisilinux/pisilife-2/master/VScode/pspec.xml

İnşa işlemi bittikten sonra:

sudo pisi it VSCode-1.3.1-1-p2-x86_64.pisi

Pisi Em

İsterseniz pisi em komutu ile direk derleyip otomatik kurmasını sağlayın:

sudo pisi em vs-code-editor
